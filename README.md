# ThutoHub

ThutoHub is a  learning app that helps learners understand concepts in their home language using AI.

## Problem
Many South African learners struggle because content is delivered only in English.

## Solution
ThutoHub uses Gemini 3 to explain Maths concepts in vernacular languages.

## Gemini Usage
- Translates explanations
- Simplifies concepts for primary learners

## Tech Stack
- Flutter
- Supabase
- Google Gemini 3 API

## Getting Started

### Importing Your Local ThutoHub Project

If you have a local ThutoHub Flutter project on your Windows machine, follow these steps to import it:

1. **Read the Import Guide**: See [IMPORT_GUIDE.md](IMPORT_GUIDE.md) for detailed instructions
2. **Use the Import Script** (Windows):
   - Run `import_project.bat` (Command Prompt), or
   - Run `import_project.ps1` (PowerShell)
3. **Or manually copy** your Flutter project files to this repository

### Running the App

1. Clone the repo
2. Import your Flutter project (if not done yet)
3. Install dependencies: `flutter pub get`
4. Add environment variables (create `.env` file)
5. Run `flutter run`
