# iOS Configuration

This directory will contain iOS-specific configuration files.

## Files to import:
- `Runner.xcodeproj/` - Xcode project files
- `Runner/Info.plist` - iOS configuration
- `Runner/AppDelegate.swift` - App delegate
- `Podfile` - CocoaPods dependencies

## Note
When importing your iOS files, make sure to:
1. Update the bundle identifier
2. Check iOS deployment target
3. Add any necessary capabilities in Xcode
